# Cookbook

End-to-end recipes that combine the client and server in realistic
analyses. Each recipe is self-contained and answers a specific
research question — copy it, adapt it, run it.

If the User Guide pages are reference material organized by
*capability*, the cookbook is organized by *task*. Use the User
Guide to understand what a function does; use the cookbook to see
how it composes with everything else.
